# `range()` and Loops Practical Scenario Questions

1. **ATM PIN System:**
   Write a program that allows a user to enter a PIN. Give the user only 3 attempts using a `for` loop.

2. **Countdown Timer:**
   Write a program to print a countdown from 10 to 1 using a `for` loop and `range()`.

3. **Multiplication Table:**
   Write a program that takes a number from the user and prints its multiplication table from 1 to 10.

4. **Shopping Cart Bill:**
   A customer has multiple product prices stored in a list. Write a program to calculate the total bill using a loop.

5. **Student Marks System:**
   Store marks of 5 subjects in a list. Write a program to calculate total marks and average using a loop.

6. **Attendance List:**
   Store names of 10 students in a list. Write a program to print each student's attendance number with their name.

7. **Bank Transaction System:**
   Given a list of deposits and withdrawals, write a program to calculate the final account balance using a loop.

8. **Password Retry System:**
   Write a program using a `while` loop that keeps asking for a password until the correct password is entered.

9. **Number Sum Calculator:**
   Write a program to find the sum of numbers from 1 to `n` using a `for` loop and `range()`.

10. **Even Number Generator:**
    Write a program to print all even numbers between 1 and 100 using `range()`.

11. **Odd Number Counter:**
    Write a program to count how many odd numbers exist between 1 and 50.

12. **Factorial Calculator:**
    Write a program to calculate the factorial of a given number using a `for` loop.

13. **File Processing Simulation:**
    Write a program that simulates processing 20 files using a loop.

14. **Login Attempts:**
    Write a program that allows a user only 5 login attempts using a loop.

15. **Reverse Number Printing:**
    Write a program to print numbers from `n` to 1 using `range()` with a negative step.

16. **Star Pattern Generator:**
    Write a program to print a star pyramid pattern using nested loops.

17. **Temperature Analyzer:**
    Given a list of daily temperatures, write a program to find the highest temperature using a loop.

18. **Shopping Discount System:**
    Write a program that applies a discount to each item price in a shopping cart using a loop.

19. **Multiples Finder:**
    Write a program to print all multiples of a given number up to 100 using `range()`.

20. **Dice Rolling Simulation:**
    Write a program that simulates rolling a dice 10 times using a loop.
