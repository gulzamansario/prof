Detailed Explanations

math.ceil()

Purpose: Returns the smallest integer that is greater than or equal to x (round up).

Syntax:

math.ceil(x)

Parameters:

  • x: Any number (int or float)

Returns: Integer value (as int)

Easy Hinglish Explanation: Socho aapke paas ek number hai jaise 2.3. Isko ceil karte hain toh yeh 3 ho jaata hai. Hamesha upar ki taraf round karta hai, chahe decimal kuch bhi ho. Agar number already integer hai toh wahi return karta hai.

Real-Life Analogy: Aap taxi booking kar rahe hain. Agar distance 2.3 km hai, toh taxi waala 3 km ka paisa lega. Woh upar round karega.

Example 1 (Basic):

import math

print(math.ceil(2.3))   # 3
print(math.ceil(2.0))   # 2
print(math.ceil(-2.3))  # -2 (less negative value)

math.comb()

Purpose: Returns the number of ways to select k items from n items without order (combinations).

Syntax:

math.comb(n, k)

Parameters:

  • n: Total number of items (non-negative integer)
  • k: Number of items to select (non-negative integer, k <= n)

Returns: Integer value (as int)

Easy Hinglish Explanation: Aapke paas n cheezon ka group hai. Aapko k cheezein choose karni hain, lekin order matter nahi karta. Kya order se farak nahi padta. Jaise 5 dost hain, 2 ko party pe bulana hai - kitne tareeke hain? Yeh function woh number batata hai.

Real-Life Analogy: Aap ice cream shop mein hain. 10 flavours hain, aapko 3 choose karne hain. Order koi matter nahi karta (vanilla-chocolate-strawberry same hai chocolate-strawberry-vanilla). Math.comb(10, 3) bataayega kitne combinations possible hain.

Example 1 (Basic):

import math

print(math.comb(5, 2))   # 10 (5 choose 2)
print(math.comb(10, 3))  # 120
print(math.comb(5, 5))   # 1 (all items selected)

math.copysign()

Purpose: Returns a float with the magnitude (value) of x but the sign of y.

Syntax:

math.copysign(x, y)

Parameters:

  • x: The number whose magnitude will be used
  • y: The number whose sign will be used

Returns: Float value

Easy Hinglish Explanation: Do numbers hain - x aur y. Yeh function x ki value leta hai, lekin uska sign (positive/negative) y se leta hai. Agar y positive hai toh result positive hoga, agar y negative hai toh result negative hoga. X ki value change nahi hoti, sirf sign change hota hai.

Real-Life Analogy: Aapko ek amount Rs. 100 mila hai (x), lekin account statement mein koi mistake se negative sign aa gaya (y). Copysign aapko bataayega ki actual amount Rs. 100 hai lekin sign y ke hisaab se laga do.

Example 1 (Basic):

import math

print(math.copysign(5.0, -3.0))   # -5.0
print(math.copysign(-5.0, 3.0))   # 5.0
print(math.copysign(2.5, -0.0))   # -2.5

math.fabs()

Purpose: Returns the absolute value of x as a float (always non-negative).

Syntax:

math.fabs(x)

Parameters:

  • x: Any number (int or float)

Returns: Float value (non-negative)

Easy Hinglish Explanation: Kisi bhi number ka positive value return karta hai. Agar number positive hai toh wahi return karega, agar negative hai toh negative sign hata dega. Hamesha float return karta hai, chahe aap int daalein.

Real-Life Analogy: Aap bank statement dekh rahe hain. Agar aapka balance -5000 hai, toh actual loss 5000 hai. Fabs woh 5000 bataayega.

Example 1 (Basic):

import math

print(math.fabs(-10))   # 10.0
print(math.fabs(10))    # 10.0
print(math.fabs(-3.5))  # 3.5

math.factorial()

Purpose: Returns the factorial of x (x! = x * (x-1) * (x-2) * ... * 1).

Syntax:

math.factorial(x)

Parameters:

  • x: Non-negative integer

Returns: Integer value

Easy Hinglish Explanation: Factorial matlab kisi number ko usse chhote saare numbers se multiply karna. Jaise 5! = 54321 = 120. Sirf positive integers ke liye kaam karta hai. 0! = 1 hota hai.

Real-Life Analogy: Aap 5 books ko shelf par kitne tareeke se arrange kar sakte hain? 5! = 120 tareeke. Yeh function woh number bataata hai.

Example 1 (Basic):

import math

print(math.factorial(5))   # 120
print(math.factorial(0))   # 1
print(math.factorial(3))   # 6

math.floor()

Purpose: Returns the largest integer less than or equal to x (round down).

Syntax:

math.floor(x)

Parameters:

  • x: Any number (int or float)

Returns: Integer value (as int)

Easy Hinglish Explanation: Kisi bhi number ko neeche ki taraf round karta hai. 2.9 ko floor karein toh 2 aayega. -2.3 ko floor karein toh -3 aayega (kyunki -3, -2 se chhota hai). Hamesha chhota integer deta hai.

Real-Life Analogy: Aap cinema hall mein hain. Ticket 500 rupees hai, aapke paas 499.50 hain toh aap nahi le sakte. Floor neeche round karega, matlab 499.50 -> 499, jo ticket ke liye kaafi nahi.

Example 1 (Basic):

import math

print(math.floor(2.9))   # 2
print(math.floor(2.0))   # 2
print(math.floor(-2.3))  # -3

math.fmod()

Purpose: Returns the remainder of x divided by y (like modulo but for floating-point numbers).

Syntax:

math.fmod(x, y)

Parameters:

  • x: Dividend
  • y: Divisor

Returns: Float value (remainder)

Easy Hinglish Explanation: Jab aap x ko y se divide karte hain, toh remainder (baki) kya bachta hai? Fmod woh bachta hua hissa return karta hai. Simple modulo (%) ki tarah, lekin yeh floats ke liye accurate hai.

Real-Life Analogy: Aap 10 rotiyan distribute kar rahe hain 3 logo mein. Har ek ko 3 rotiyan milengi (9 gyi), 1 bachegi. Fmod(10, 3) = 1.0.

Example 1 (Basic):

import math

print(math.fmod(10, 3))    # 1.0
print(math.fmod(7.5, 2.5)) # 0.0
print(math.fmod(10, 4))    # 2.0

math.frexp()

Purpose: Returns the mantissa (m) and exponent (e) of x such that x = m * 2**e.

Syntax:

math.frexp(x)

Parameters:

  • x: Any number (int or float)

Returns: Tuple (mantissa, exponent) where mantissa is float between 0.5 and 1, exponent is int

Easy Hinglish Explanation: Kisi bhi number ko scientific notation mein break karta hai. Jaise 10 = 0.625 * 16 (0.625 * 2**4). Yeh function (0.625, 4) return karega. Isse computers numbers store karte hain.

Real-Life Analogy: Aap weight ko alag tareeke se likhna chahte hain. Jaise 100 kg = 1 * 100, lekin yahan 2 ki power mein likhte hain. Frexp yeh conversion karta hai.

Example 1 (Basic):

import math

print(math.frexp(10))   # (0.625, 4) because 0.625 * 2**4 = 10
print(math.frexp(8))    # (0.5, 4) because 0.5 * 2**4 = 8

math.fsum()

Purpose: Returns an accurate sum of all numbers in an iterable (avoids floating-point errors).

Syntax:

math.fsum(iterable)

Parameters:

  • iterable: Any iterable (list, tuple, etc.) containing numbers

Returns: Float value (sum)

Easy Hinglish Explanation: Normal sum() function floating-point numbers ko add karte time small errors create kar sakta hai. Fsum un errors ko avoid karta hai aur accurate sum deta hai. Especially important jab bahut saare decimal numbers add kar rahe ho.

Real-Life Analogy: Aap 1000 transactions ka total nikaal rahe hain. Har transaction mein paise hain (decimal). Fsum ensure karega ki total exact aaye, koi 0.0001 ka error na ho.

Example 1 (Basic):

import math

print(math.fsum([0.1, 0.2, 0.3]))   # 0.6
print(math.fsum([1.0, 2.0, 3.5]))    # 6.5
print(math.fsum([0.1] * 10))          # 1.0 (accurate)

math.gcd()

Purpose: Returns the greatest common divisor (GCD) of x and y.

Syntax:

math.gcd(x, y)

Parameters:

  • x: First integer
  • y: Second integer

Returns: Integer value (GCD)

Easy Hinglish Explanation: Do numbers lo. Un dono ko divide karne wala sabse bada number kya hai? GCD woh number bataata hai. Jaise 12 aur 18 dono 6 se divide ho sakte hain, aur 6 se bada koi nahi, toh GCD = 6.

Real-Life Analogy: Aapke paas 12 apples aur 18 oranges hain. Aapko dono ko equal groups mein distribute karna hai. Sabse bada group size kya ho sakta hai? GCD(12, 18) = 6, matlab 6 apples ke 2 groups aur 6 oranges ke 3 groups.

Example 1 (Basic):

import math

print(math.gcd(12, 18))   # 6
print(math.gcd(100, 35))  # 5
print(math.gcd(17, 19))   # 1 (coprime)

math.isclose()

Purpose: Checks if two numbers are close to each other within a tolerance.

Syntax:

math.isclose(a, b, *, rel_tol=1e-09, abs_tol=0.0)

Parameters:

  • a: First number
  • b: Second number
  • rel_tol: Relative tolerance (default 1e-09)
  • abs_tol: Absolute tolerance (default 0.0)

Returns: Boolean (True/False)

Easy Hinglish Explanation: Do numbers compare karte hain. Agar wo ek doosre ke bahut close hain (toleration ke andar) toh True return karta hai, nahi toh False. Computers mein floating-point numbers exact nahi hote, isliye exact equal check karna mushkil hai. Isclose se hum check kar sakte hain ki numbers almost equal hain ya nahi.

Real-Life Analogy: Aap do measuring tapes se length measure kar rahe hain. Ek ne 10.0001 cm dikhaya, doosre ne 10.0002 cm. Exact same nahi hain, lekin almost same hain. Isclose bolega ki yeh close hain.

Example 1 (Basic):

import math

print(math.isclose(0.1 + 0.2, 0.3))           # True
print(math.isclose(1.0, 1.0000000001))        # True
print(math.isclose(1.0, 1.0001, rel_tol=1e-3)) # True (1e-3 = 0.001)

math.isfinite()

Purpose: Checks if x is a finite number (not infinity and not NaN).

Syntax:

math.isfinite(x)

Parameters:

  • x: Any number

Returns: Boolean (True/False)

Easy Hinglish Explanation: Check karta hai ki number finite hai ya nahi. Infinite aur NaN (Not a Number) finite nahi hain. Baaki sab finite hain.

Real-Life Analogy: Aap account balance check kar rahe hain. Agar balance ek actual number hai toh finite hai. Agar NaN (error) ya infinite (unlimited) hai toh finite nahi hai.

Example 1 (Basic):

import math

print(math.isfinite(10))          # True
print(math.isfinite(float('inf')))# False
print(math.isfinite(float('nan')))# False

math.isinf()

Purpose: Checks if x is positive or negative infinity.

Syntax:

math.isinf(x)

Parameters:

  • x: Any number

Returns: Boolean (True/False)

Easy Hinglish Explanation: Check karta hai ki number infinite hai ya nahi. Positive infinity (+inf) ya negative infinity (-inf) dono True honge.

Real-Life Analogy: Aap speed check kar rahe hain. Agar speed unlimited (infinite) hai toh isinf True return karega. Actual speed hai toh False.

Example 1 (Basic):

import math

print(math.isinf(float('inf')))    # True
print(math.isinf(float('-inf')))   # True
print(math.isinf(10))               # False

math.isnan()

Purpose: Checks if x is Not-a-Number (NaN).

Syntax:

math.isnan(x)

Parameters:

  • x: Any number

Returns: Boolean (True/False)

Easy Hinglish Explanation: Check karta hai ki number NaN (Not a Number) hai ya nahi. NaN tab aata hai jab koi invalid mathematical operation ho, jaise 0/0.

Real-Life Analogy: Aapne kisi calculation mein mistake ki hai, result undefined aa gaya. Isnan check karega ki result undefined hai ya nahi.

Example 1 (Basic):

import math

print(math.isnan(float('nan')))    # True
print(math.isnan(10))               # False
print(math.isnan(0/0))              # True (in Python, 0/0 gives error, but some operations give NaN)

Syntax:

math.isqrt(x)

Parameters:

  • x: Non-negative integer

Easy Hinglish Explanation: Kisi number ka square root nikaalo, lekin sirf integer part lo. Jaise 10 ka square root 3.16 hai, isqrt 3 dega. Yeh hamesha integer return karta hai.

Real-Life Analogy: Aapke paas 10 tiles hain, aapko square banane hain. Sabse bada square kitna bada ho sakta hai? 3x3 = 9 tiles, 4x4 = 16 tiles nahi ho sakta. Toh isqrt(10) = 3.

import math

print(math.isqrt(10))   # 3

math.ldexp()

Purpose: Returns x * (2**i), the inverse of frexp.

Syntax:

math.ldexp(x, i)

Parameters:

  • x: Mantissa (float)
  • i: Exponent (integer)

Returns: Float value

Easy Hinglish Explanation: Frexp ne number ko break kiya tha (m, e) mein. Ldexp reverse karta hai - mantissa ko 2 ki power se multiply karke original number return karta hai.

Real-Life Analogy: Aapne 100 kg ko break kiya 1 * 100 mein. Ldexp wapas 100 kg banayega.

Example 1 (Basic):

import math

m, e = math.frexp(10)
print(math.ldexp(m, e))   # 10.0
print(math.ldexp(0.5, 4)) # 8.0

math.lcm()

Purpose: Returns the least common multiple (LCM) of the given integers.

Syntax:

math.lcm(*integers)

Parameters:

  • *integers: Variable number of integers (2 or more)

Returns: Integer value (LCM)

Easy Hinglish Explanation: Do ya zyada numbers lo. Un sab numbers ka sabse chhota common multiple kya hai? LCM woh number bataata hai. Jaise 4 aur 6 ka LCM 12 hai (12 dono se divide ho jaata hai aur 12 se chhota koi bhi dono se nahi divide hota).

Real-Life Analogy: Aapko 4 aur 6 logo ka time slot match karna hai. Kitne minute baad dono wapas milenge? LCM(4, 6) = 12 minutes.

Example 1 (Basic):

import math

print(math.lcm(4, 6))       # 12
print(math.lcm(2, 3, 5))    # 30
print(math.lcm(10, 15))     # 30

math.perm()

Purpose: Returns the number of ways to select and arrange k items from n items (permutations).

Syntax:

math.perm(n, k=None)

Parameters:

  • n: Total number of items (non-negative integer)
  • k: Number of items to select and arrange (optional, default n)

Returns: Integer value (number of permutations)

Easy Hinglish Explanation: Aapke paas n items hain, aapko k items choose karna hai aur unka order matter karta hai. Kitne tareeke hain? Yehi permutations hain. Agar k nahi diya toh n items ka full arrangement count karega (n!).

Real-Life Analogy: 3 racers hain, gold, silver, bronze medal kaun jeetega? Order matter karta hai. Perm(3, 3) = 6 possible rankings.

import math

print(math.perm(5, 2))   # 20 (5*4)
print(math.perm(3, 3))   # 6 (3*2*1)

Syntax:

math.nextafter(x, y)

Parameters:

  • x: Starting number
  • y: Direction (towards this number)

Returns: Float value (the next representable float)

Easy Hinglish Explanation: Computers mein floats limited precision mein store hote hain. Is function se aap x ke aage wala next possible float value le sakte ho, y ki direction mein. Agar x = y, toh wahi return karega.

Real-Life Analogy: Socho aap ek scale pe hain jahan sirf specific decimal values possible hain. Nextafter aapko agla possible value bataayega.

math.remainder()

Purpose: Returns the IEEE 754-style remainder of x divided by y.

Syntax:

math.remainder(x, y)

Parameters:

  • x: Dividend
  • y: Divisor

Returns: Float value (remainder)

Easy Hinglish Explanation: Fmod ki tarah hi remainder deta hai, lekin yeh IEEE 754 standard follow karta hai. Isme sign aur value thoda alag ho sakta hai. Remainder x - n*y hota hai, jahan n sabse close integer hai x/y ka.

Real-Life Analogy: Time calculation mein kaam aata hai. Jaise 10 ko 3 se divide karte hain, 3.33 aata hai. Remainder bataayega kitna bacha.

Example 1 (Basic):

import math

print(math.remainder(10, 3))   # 1.0 (since 10 - 3*3 = 1)
print(math.remainder(10, 4))   # 2.0 (10 - 4*2 = 2)
print(math.remainder(7, 2.5))  # -0.5 (7 - 2.5*3 = -0.5)

import math

Real-Life Analogy: Bahut chhote interest rates calculate kar rahe ho, expm1 use karo taaki accurate result aaye.

Example 1 (Basic):

math.log()

Purpose: Returns the natural logarithm of x (or logarithm with specified base).

Syntax:

math.log(x, base)

Parameters:

  • x: Number to take logarithm of
  • base: Base (optional, default is e)

Returns: Float value

Easy Hinglish Explanation: Logarithm reverse of exponent hai. Jaise log(100) = 2 (10^2 = 100). Default natural log (base e) hota hai, lekin aap base specify kar sakte hain.

Real-Life Analogy: Earthquake magnitude Richter scale pe log scale hai. Magnitude 7 ka earthquake, magnitude 6 se 10x zyada powerful hai. Log se yeh calculate hota hai.

Example 1 (Basic):

import math

print(math.log(2.718))        # ~1.0 (natural log)
print(math.log(100, 10))      # 2.0 (log base 10)
print(math.log(8, 2))         # 3.0 (log base 2)

Key Takeaways

  1. Math module mein 50+ functions hain jo mathematical operations provide karte hain.

  2. Trigonometric functions radians use karte hain, degrees nahi. Radians/degrees convert karna yaad rakhein.

  3. Floating-point precision ke liye special functions hain: math.fsum(), math.isclose(), math.nextafter(), etc.

  4. Constants jaise pi, e, tau, inf, nan readily available hain.

  5. Number theory functions: gcd, lcm, isqrt, factorial, comb, perm.

  6. Always import math module use karne se pehle.

  7. Most functions float return karte hain, except jo specifically int return karte hain (ceil, floor, trunc, factorial, etc.)
